# @44North Prettier Config

The standard 44 North [Prettier](https://prettier.io/) Configuration.

## Install

```
yarn add prettier @44north/prettier-config --dev
```

## Configure

```js
// prettier.config.js
const config = require("@44north/prettier-config");

module.exports = {
    ...config
};
```
